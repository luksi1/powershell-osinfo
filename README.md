# powershell-osinfo

## Description

Get disk and service information from the Common Infrastructure Model (CIM) via PowerShell

## Usage

### Disk information

Get-DiskInformation -ComputerName MYCOMPUTER

### Service Information

Get-ServiceInformation -ComputerName MYCOMPUTER
